# 土壤质量指数 (SQI) 影响因素分析

这是一个用于分析土壤质量指数(SQI)影响因素的 Python 数据分析项目，通过机器学习方法识别不同地区影响土壤质量的关键因素。

## 📋 项目简介

本项目旨在通过数据挖掘和机器学习技术，分析不同地区（广西、江西）的土壤质量指数及其影响因素，为农业生产和土壤管理提供科学依据。

### 核心功能

- **数据预处理**: 自动加载和清理土壤实验数据
- **分组建模**: 按地区（广西、江西）分别建立随机森林回归模型
- **特征重要性分析**: 识别影响土壤质量的关键因素
- **处理方式评估**: 比较不同处理方式对 SQI 的效果
- **可视化报告**: 生成详细的文本分析报告

## 🔧 技术栈

- **Python**: 核心编程语言
- **Pandas**: 数据处理与分析
- **NumPy**: 数值计算
- **Scikit-learn**: 机器学习（随机森林回归、标签编码）
- **Matplotlib & Seaborn**: 数据可视化
- **XlsxWriter/Excel**: Excel 文件读取

### 使用方法

1. 准备数据文件 `cleaned_data.xlsx`，包含以下列：
   - `treatment`: 处理方式（分类变量）
   - `pH`: 酸碱度
   - `CEC`: 阳离子交换量
   - `Ex.Ca`: 交换性钙
   - `Ex.Mg`: 交换性镁
   - `AP`: 有效磷
   - `AK`: 有效钾
   - `pHBC`: pH 缓冲容量
   - `SQI`: 土壤质量指数（目标变量）
   - `region`: 地区（广西/江西）

2. 修改 `main()` 函数中的文件路径：

```python
file_path = '你的数据文件路径/cleaned_data.xlsx'
```

3. 运行脚本：

```bash
python soil_analysis.py
```

## 📊 分析流程

1. **数据加载与预处理**
   - 读取 Excel 数据
   - 检查缺失值
   - 对分类变量进行标签编码

2. **地区分组**
   - 按广西、江西地区拆分数据
   - 统计各地区样本数量

3. **模型训练**
   - 为每个地区训练独立的随机森林回归模型
   - 评估模型性能（R² 得分）

4. **特征重要性分析**
   - 识别各地区影响 SQI 的关键因素
   - 生成 CSV 格式的特征重要性报告

5. **处理方式评估**
   - 对比不同处理方式对 SQI 的影响
   - 生成处理方式排名

6. **报告生成**
   - 输出控制台分析结果
   - 保存详细的文本报告

2. 数据文件应包含所有必需的列

4. 中文字体设置适用于 Windows 系统，Mac/Linux 可能需要调整

## 🔬 算法说明

- **随机森林回归 (Random Forest Regressor)**: 使用 100 棵决策树，能够有效处理非线性关系和高维特征，自动评估特征重要性。
- **标签编码 (Label Encoder)**: 将分类变量转换为数值，便于模型训练。
